# Handoff Skill Repository

This repository contains the standalone `handoff` skill authored by Verne.

## Positioning

- Public standalone skill repo
- For Claude Code and OpenCode
- Not advertised as a Codex skill

## Structure

- `SKILL.md` - skill entrypoint
- `SKILL-setup.md` / `SKILL-commands.md` - setup and command docs
- `scripts/` - runtime helpers and tests
- `worker/` - worker config
